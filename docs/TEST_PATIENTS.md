# 测试病例 · 三个典型场景

> 配套 `TEST_PROTOCOL_NSCLC_PD1.docx` 使用。
> 三位患者分别**清楚符合**、**清楚排除**、**边缘场景**,用来验证 Eligibility engine 是否会给出合理的命中 / 排除 / 置信度。

---

## 病人 A — 张三 · 清楚符合

> 期望:eligibility engine 命中 8/9 入选条目,0 排除条目,**高置信度候选**。
> 这是验证 "可以加入研究" 提醒功能的标准案例。

### Phase 2 录入(新增病人弹窗)

| 字段 | 值 |
|---|---|
| 姓名首字母 | `Z.S` |
| MRN | `MRN-001-2026` (可选) |
| 性别 | M |
| 年龄段 | 60s |

### Phase 2 后续 — 在记忆 tab / 问诊里录入临床信息

**初诊主诉(粘到问诊里发一条消息即可,LLM 会抽进 clinical_graph)**:

```
65 岁男性,因「咳嗽 2 月,加重伴气紧 2 周」就诊。

外院 PET-CT (2026-05-18):
- 右肺上叶尖段不规则占位 3.2×2.8cm,SUVmax 12.5
- 同侧纵隔 4R、7 组淋巴结肿大,SUVmax 8.2
- 全身余处未见 FDG 高代谢

支气管镜活检 (2026-05-22): 肺腺癌,中分化。

EGFR/ALK/ROS1: 全阴性 (外院 NGS,2026-05-30)。
临床分期: cT2bN2M0 IIIB 期 (AJCC v8)
胸外科 MDT 评估: 不可切除。

ECOG: 1 分
肺功能: FEV1 2.1 L (78% 预计值),DLCO 65% 预计值。
血常规: WBC 6.2, ANC 4.1, PLT 215, Hb 132。
肝肾: ALT 28, AST 25, Cr 78, Bili 11。

既往史: 高血压 5 年,氨氯地平控制。无放化疗史。
肿瘤标志物 NSE 正常,CEA 32 ng/ml ↑。
```

**关键命中条目**(eligibility engine 应该全部识别):
- ✅ 年龄 65,在 18-75 区间
- ✅ NSCLC 腺癌
- ✅ IIIB 期 (cT2bN2M0)
- ✅ 未接受抗肿瘤治疗
- ✅ ECOG 1
- ✅ 至少一个可测量病灶 (3.2×2.8cm)
- ✅ FEV1 2.1 L, DLCO 65%(均达标)
- ✅ 实验室全部达标
- ✅ EGFR/ALK/ROS1 阴性 (避开排除条目 2)

## 病人 B — 李四 · 清楚排除

> 期望:eligibility engine **不应该**把李四列为该研究候选。如果出现就是 bug。
> 应该命中多个排除条目(SCLC、既往放疗、ECOG 高、年龄超限)。

### Phase 2 录入

| 字段 | 值 |
|---|---|
| 姓名首字母 | `L.S` |
| MRN | `MRN-002-2026` |
| 性别 | F |
| 年龄段 | 70s |

### 初诊主诉

```
78 岁女性,反复咳嗽 3 月,近 2 周咯血。

胸部 CT (2026-06-05):
- 右肺门巨大占位 6.8×5.2cm,纵隔多组淋巴结融合
- 双肺多发结节,最大 1.2cm(左下叶),考虑肺内转移
- 肝右叶可见 2.5cm 低密度灶,考虑转移瘤

支气管镜活检 (2026-06-08): 小细胞肺癌 (SCLC),
化疗敏感型,Ki-67 90%。

临床分期: 广泛期 SCLC (ES-SCLC)
ECOG: 2 分

既往史:
- 2024 年因左肺鳞癌行 SBRT 60 Gy/8 次。
- 类风湿关节炎,长期甲氨蝶呤 + 强的松 10 mg qd。
- HBsAg 阳性,HBV-DNA 5.2×10^4 IU/mL。

血常规: WBC 4.1, ANC 1.8, PLT 88, Hb 95。
肝肾: ALT 65, AST 72, Cr 95, Bili 18。
```

**关键排除条目命中**:
- ❌ 年龄 78 超过 75 上限(入选 1 不通过)
- ❌ 小细胞肺癌(排除 1)
- ❌ 既往胸部放疗史(排除 4)
- ❌ 自身免疫病 + 长期免疫抑制剂(排除 5)
- ❌ 活动性 HBV(排除 7)
- ❌ ECOG 2(排除 11 + 入选 5 不通过)
- ❌ 肝功能异常(ALT/AST 超 2.5×ULN)
- ❌ 影像/病理提示远处转移,不再是 III 期(入选 3 不通过)

**期望 UI 行为**:候选清单**不应该**出现李四。如果出现,大概率是分期识别错(把 ES-SCLC 当成 NSCLC IIIB)。

## 病人 C — 王五 · 边缘场景

> 期望:eligibility engine **可能**列为候选,但**置信度应该偏低**或标红一个排除点让医生人工 review。
> 这是测试系统是否有"灰色地带审慎"的设计能力。

### Phase 2 录入

| 字段 | 值 |
|---|---|
| 姓名首字母 | `W.W` |
| MRN | `MRN-003-2026` |
| 性别 | M |
| 年龄段 | 50s |

### 初诊主诉

```
58 岁男性,体检发现右肺占位 1 月。

PET-CT (2026-06-12):
- 右肺下叶背段 4.5×3.1cm 占位,SUVmax 9.8
- 同侧肺门、纵隔 4R/7 组淋巴结,SUVmax 6.5
- 双肾、肝、骨未见异常

CT 引导穿刺 (2026-06-14): 肺鳞癌,低分化。
EGFR/ALK/ROS1 检测中(尚未出报告)。

临床分期: cT2bN2M0 IIIA 期
胸外科评估: 技术上可切但合并症高风险,推荐 cCRT。

ECOG: 1 分
肺功能: FEV1 1.4 L (52% 预计值)边缘达标
DLCO 41% 预计值,临界值。

既往史:
- 慢阻肺 GOLD 2 级,长期布地奈德/福莫特罗吸入。
- 5 年前甲状腺乳头状癌行甲状腺全切 + I-131 治疗,
  目前无复发证据。

血常规、肝肾基本正常。
HBsAg 阴性,HCV-Ab 阴性。
```

**灰色地带**:
- ⚠️ 年龄 58 ✓
- ⚠️ 鳞癌 ✓ (NSCLC)
- ⚠️ IIIA 期 ✓ (入选 3 接受)
- ⚠️ MDT 推荐 cCRT 而非手术 (技术上可切但临床决定不切)
- ⚠️ FEV1 52% **刚过线**,DLCO 41% **刚过线**
- ⚠️ EGFR 报告尚未出 — **入排条目 2 "已知 EGFR 突变" 当前"未知",通常算"可入" 但应 flag**
- ⚠️ 既往甲状腺癌 + I-131(5 年前)— **触发排除条目 10 "既往 5 年内其他恶性肿瘤"**,需要算从治疗结束还是发现日;**严格算 5 年还是 4.X 年是灰色**

**期望 UI 行为**:
- 出现候选,**但置信度中等**(70% 以下)
- CandidateCard 应该明显标出"既往甲状腺癌临界 5 年"、"EGFR 待测"两个 yellow flag
- 医生应该能在 review 之后选择"邀请入组"或"待定"或"跳过"

如果系统简单地把王五算成命中 8/8 跟张三同等置信度,那 eligibility engine 太粗糙;如果直接排除王五,又太保守。**理想:候选出现 + 置信度可疑 + 标注关键灰色点 + 把决策权留给医生**(符合设计文档 anti-pattern 第 3 条 "不让 AI 替医生拍板")。

## 测试动线建议

按这个顺序录,验证完整流程:

1. **Phase 1**:导入 `TEST_PROTOCOL_NSCLC_PD1.docx` → 研究建立
2. **Phase 2 + 3**:先录张三 → 切到研究 / 入排清单 → 看到张三候选 → 邀请入组 #001
3. **Phase 2 + 3**:录李四 → 入排清单 → 应该**没有**李四出现
4. **Phase 2 + 3**:录王五 → 入排清单 → 看到王五候选但有 yellow flag
5. **Phase 5**:回到张三,manual record 一条 G3 肺炎 AE → 标 DLT → 看 stop-rule 条变化
6. **Phase 7**:切张三/王五验证问诊 session 不混
